Overview of Demos

The demos in this section include some walk-through of the compose.mk capabilities. Most demos are covering library use of compose.mk and basically building extensions, but some demos will also cover aspects of stand-alone / tool mode. When we're scripting, you're usually reading ready-to-run code that's available in the repository with related discussion, and the code itself runs as part of the test-suite.

Caveats

Before we get into a guided tour, this is a good place for the obligatory disclaimer.

Many of the demonstrations here are serious, but it has to be said that some are frankly nuts, and most likely no one will thank you for introducing these techniques into real projects! Now, with that stern warning out of the way..
